## Overview

SAPPDiscordBot listens for **server events** generated by
the [SAPP Lua script](https://github.com/Chalwk/HALO-SCRIPT-PROJECTS/blob/master/sapp/utility/discord.lua), processes
them, and sends rich
notifications to Discord. It supports **multiple servers**, offers a **GUI** for configuration, and is optimized for
performance.

---

## Features

* Multi-server support
* Rich Discord embeds
* Real-time event processing
* Java Swing GUI with tray integration
* Configurable notifications
* High performance text event format
* Detailed logging
* Easy configuration panel

---

## System Requirements

* Java 17 or later
* 512MB RAM
* 50MB storage
* Windows or Linux
* Halo server running SAPP
* [Discord Bot Token](https://discord.com/developers/applications)

---

### Download and Installation

1. Download the [latest release](https://github.com/Chalwk/SAPPDiscordBot/releases) from the repository
2. Run `SAPPDiscordBot.exe`
3. Enter your Discord Bot Token
4. Set your watch directory
5. Install the `discord.lua` script on your server

---

## Supported Event Types

| Category          | Events                                                  |
|-------------------|---------------------------------------------------------|
| Game Events       | event_start, event_end, event_map_reset                 |
| Player Events     | event_join, event_leave, event_spawn, event_team_switch |
| Chat and Commands | event_chat, event_command                               |
| Score Events      | event_score                                             |
| Death Events      | event_death                                             |
| Admin Events      | event_login, event_snap                                 |
